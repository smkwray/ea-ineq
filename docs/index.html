Fiscal Transfers, Poverty, Inequality, and Consumption Composition

Main takeaway:

This archive is the project-specific public layer for an inequality-and-poverty research workflow built on econark’s R pipeline. It packages the compact project inputs, the native DASS and DFLMX result trees, and a readable front end for the evidence that matters most.

How to read this archive
DFLMX
The screening layer. It ranks LP impulse-response rows, applies BH-FDR correction (q ≤ 0.10), and runs episode, lead, and confirmatory diagnostics.
DASS
The estimation layer. It preserves estimator-specific outputs for the same treatment-outcome-horizon rows, so you can judge whether a headline row has broad backing or is LP-only.
LP
Local projections — horizon-by-horizon regressions asking what happens 0, 1, 2, 4, or 8 quarters after a shock.
DML
Debiased machine learning — a flexible regression that removes what controls predict from both treatment and outcome using sample splitting.
TMLE
Targeted maximum likelihood estimation — a “high shock versus low shock” comparison that provides an independent cross-check.
CF / LP-IV / DML-IV
Additional DASS estimators: causal forests, LP with instrumental variables, and DML with instrumental variables.
Robust row
A DFLMX finding with q ≤ 0.10 after FDR correction.
Screening-only
A row that survives the screening stack but still lacks IV/negative-control support for a stronger confirmatory claim.
