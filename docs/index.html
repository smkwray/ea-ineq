Fiscal Transfers, Poverty, Inequality, and Consumption Composition

Main takeaway:

This archive is the project-specific public layer for an inequality-and-poverty research workflow built on econark's R pipeline. It packages the compact project inputs, the native DASS and DFLMX result trees, and a readable front end for the parts of the archive that matter most.

The site is organized the same way the research logic is organized. DFLMX provides the headline screen and robustness stack. DASS preserves the estimator layer underneath it. The point of the archive is to show both: what survived the screen, and how much estimator support sits underneath those surviving rows.

How to read this archive
DFLMX
The screening layer. It ranks LP impulse-response rows, applies FDR correction, and runs episode, lead, and confirmatory diagnostics.
DASS
The estimation layer. It keeps the estimator-specific outputs used to judge whether a headline row has LP-only support or broader estimator backing.
Robust row
A DFLMX finding with q ≤ 0.10 after multiple-testing correction.
Estimator support
Whether the same treatment-outcome-horizon row points in the same direction, and reaches nominal significance, across LP, DML, TMLE, or related DASS estimators.
Screening-only
A row that survives the main screening stack but still lacks enough IV/negative-control support for a stronger confirmatory claim.
